
[size=20][b]Обзор[/b][/size]

События — это информационные уведомления, передаваемые между объектами на сервере. Многие из них также передаются с сервера клиенту. Они определены в .res файлах в папке hl2/resource и папках resource конкретных модов. Для базового списка см. [url=https://cs2.poggu.me/dumped-data/game-events/]Source2 Game Events[/url].

Важно отметить несколько концепций о событиях:

[list]
[*]Они почти всегда информационные. То есть блокирование player_death не остановит смерть игрока. Это может заблокировать сообщение в HUD или консоли или что-то еще незначительное.
[*]Они почти всегда используют userids вместо слотов клиентов.
[*]То, что событие находится в файле ресурсов, не означает, что оно когда-либо вызывается или работает так, как вы ожидаете. Моды печально известны тем, что неправильно документируют функциональность своих событий.
[/list]

[quote][b]Info:[/b] [b]Новое[/b]: Класс [font=monospace]EventInfo[/font] предоставляет более чистый способ работы с игровыми событиями, с автоматическим управлением ресурсами и типизированными методами.[/quote]

[size=20][b]Добавление слушателя событий[/b][/size]

Вы можете привязать слушатели событий в [font=monospace]OnPluginStart[/font] (или где угодно).

[spoiler=c#][/code]csharp
  using Plugify;
  using static s2sdk.s2sdk;

  public unsafe class Sample : Plugin
  {
      public void OnPluginStart()
      {
          HookEvent("player_death", (string name, nint event, bool dontBroadcast) =>
          {
              int userid = GetEventInt(event, "userid");
              PrintToServer("event called.\n");
              return ResultType.Continue;
          }, HookMode.Post);
      }
  }
  [code][/spoiler]

[size=20][b]Доступ к параметрам событий[/b][/size]

[spoiler=С классами (рекомендуется)]
Доступ к параметрам событий с использованием типизированных методов на экземпляре EventInfo:
  [font=monospace]`[/font]csharp
  // Чтение параметров событий
  int userid = event.GetInt("userid");
  float damage = event.GetFloat("damage");
  string weaponName = event.GetString("weapon");
  bool headshot = event.GetBool("headshot");

  // Получение ссылок на сущности
  int playerController = event.GetPlayerController("userid");
  int playerPawn = event.GetPlayerPawn("userid");
  int entity = event.GetEntity("entityid");
  [font=monospace]`[/font]

  Установка параметров событий столь же проста:
  [font=monospace]`[/font]csharp
  // Установка параметров событий
  event.SetInt("userid", 123);
  event.SetFloat("damage", 50.0f);
  event.SetString("weapon", "ak47");
  event.SetBool("headshot", true);

  // Установка ссылок на сущности
  event.SetPlayerController("userid", controllerHandle);
  event.SetEntity("entityid", entityHandle);
  [font=monospace]`[/font]
[/spoiler]

[size=20][b]Создание и запуск событий[/b][/size]

[spoiler=С классами (рекомендуется)]
Создавайте и запускайте пользовательские события используя класс EventInfo:
  [font=monospace]`[/font]csharp
  // Создание нового события
  using (var customEvent = new EventInfo("player_hurt"))
  {
      // Установка параметров события
      customEvent.SetInt("userid", 1);
      customEvent.SetInt("attacker", 2);
      customEvent.SetInt("health", 50);
      customEvent.SetString("weapon", "ak47");

      // Запуск для всех игроков
      customEvent.Fire();

      // Или запуск для конкретного игрока
      customEvent.FireToClient(playerSlot);
  } // Автоматическая очистка
  [font=monospace]`[/font]
[/spoiler]

[spoiler=Без классов]
Создавайте и запускайте пользовательские события используя явные функции:
  [font=monospace]`[/font]csharp
  // Создание нового события
  nint customEvent = CreateEvent("player_hurt");

  // Установка параметров события

  SetEventInt(customEvent, "attacker", 2);
  SetEventInt(customEvent, "health", 50);

  // Или запуск для конкретного игрока

[size=20][b]Отмена события[/b][/size]

В хуке pre-события вы можете предотвратить передачу события другим плагинам, вернув [font=monospace]ResultType.Handled[/font] или [font=monospace]ResultType.Stop[/font].

[spoiler=С классами (рекомендуется)][/code]csharp
  public void OnPluginStart()
  {
      HookEvent("player_death", OnPlayerDeathPre, HookMode.Pre);
  }

  private static ResultType OnPlayerDeathPre(string name, nint @event, bool dontBroadcast)
  {
      EventInfo event = new EventInfo(@event);
      int victim = event.GetInt("userid");

      // Отмена события смерти для игроков-администраторов
      if (IsPlayerAdmin(victim))
      {
          PrintToServer("Prevented admin death event\n");
          return ResultType.Handled;
      }

      return ResultType.Continue;
  }
  [code][/spoiler]
