Plugify comes with a collection of powerful plugins that extend its capabilities and provide essential functionality for developers and modders. These plugins offer features ranging from configuration management to dynamic function hooking and Source 2 engine integration.

### Available Plugins

::card-group
::card
---
title: Config System
icon: lucide:settings
to: /plugins/config-system/features
---
Manage configuration files for plugins with multi-format support (JSON, XML, KeyValues). Automatically merges user configurations with plugin updates.
::

::card
---
title: Admin System
icon: lucide:shield
to: /plugins/admin-system/features
---
A comprehensive admin system for managing user permissions and access control in your Plugify-based applications.
::

::card
---
title: PolyHook
icon: lucide:link
to: /plugins/polyhook/features
---
Dynamic function hooking with pre and post callbacks. Supports x86/x64 on Windows and Linux using AsmJit and Capstone.
::

::card
---
title: DynCall
icon: lucide:phone-call
to: /plugins/dyncall/features
---
Call C functions dynamically at runtime. Cross-platform library for manual argument binding and programmatic function calls.
::

::card
---
title: S2SDK
icon: lucide:gamepad-2
to: /plugins/s2sdk/features
---
Specialized toolkit for Source 2 engine plugin development. Create custom gameplay features, UI elements, and game modifications.
::
::

### Why Use Plugify Plugins?

- **Modular Design**: Each plugin is self-contained and can be used independently or combined with others
- **Cross-Platform**: Most plugins support Windows, Linux, and macOS
- **Well-Documented**: Comprehensive guides and examples for each plugin
- **Community-Driven**: Built with input from the community and regularly updated
- **Easy Integration**: Simple installation and configuration process

### Need More Functionality?

If you need a plugin that doesn't exist yet, you can:
- Request a new plugin in the [Feature Requests](/community-support/feature-requests) section
- Create your own plugin using any [supported language](/languages/languages)
- Contribute to existing plugins on [GitHub](https://github.com/untrustedmodders/)
