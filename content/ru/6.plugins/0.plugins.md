Plugify поставляется с коллекцией мощных плагинов, которые расширяют его возможности и предоставляют необходимую функциональность для разработчиков и моддеров. Эти плагины предлагают функции от управления конфигурациями до динамического перехвата функций и интеграции с движком Source 2.

### Доступные плагины

::card-group
::card
---
title: Config System
icon: lucide:settings
to: /ru/plugins/config-system/features
---
Управление файлами конфигурации для плагинов с поддержкой различных форматов (JSON, XML, KeyValues). Автоматически объединяет пользовательские конфигурации с обновлениями плагинов.
::

::card
---
title: Admin System
icon: lucide:shield
to: /ru/plugins/admin-system/features
---
Комплексная система администрирования для управления правами пользователей и контроля доступа в приложениях на базе Plugify.
::

::card
---
title: PolyHook
icon: lucide:link
to: /ru/plugins/polyhook/features
---
Динамический перехват функций с обратными вызовами до и после выполнения. Поддержка x86/x64 на Windows и Linux с использованием AsmJit и Capstone.
::

::card
---
title: DynCall
icon: lucide:phone-call
to: /ru/plugins/dyncall/features
---
Динамический вызов C-функций во время выполнения. Кроссплатформенная библиотека для ручного связывания аргументов и программного вызова функций.
::

::card
---
title: S2SDK
icon: lucide:gamepad-2
to: /ru/plugins/s2sdk/features
---
Специализированный набор инструментов для разработки плагинов для движка Source 2. Создавайте пользовательские игровые функции, элементы интерфейса и модификации.
::
::

### Зачем использовать плагины Plugify?

- **Модульный дизайн**: Каждый плагин самодостаточен и может использоваться независимо или в сочетании с другими
- **Кроссплатформенность**: Большинство плагинов поддерживают Windows, Linux и macOS
- **Хорошая документация**: Подробные руководства и примеры для каждого плагина
- **Поддержка сообщества**: Созданы с учетом мнения сообщества и регулярно обновляются
- **Простая интеграция**: Простой процесс установки и настройки

### Нужна дополнительная функциональность?

Если вам нужен плагин, которого еще не существует, вы можете:
- Запросить новый плагин в разделе [Запросы функций](/ru/community-support/feature-requests)
- Создать свой собственный плагин на любом [поддерживаемом языке](/ru/languages/languages)
- Внести вклад в существующие плагины на [GitHub](https://github.com/untrustedmodders/)
